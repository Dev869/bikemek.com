About Us<\/h1>\n\n\n\n

The BikeMek Studio has 7 functional workstands each with their own set of basic tools where you can work on your own bike along with three Park Shop Wheel truing stands. The studio also has major equipment you will need to perform larger repairs.<\/p>\n\n\n\n

Most multi-week classes include a pass to use the studio while classes are not in session or staff are not working on repairs. If you are already skilled and would prefer not to take a class, you can reserve a work stand by the hour. All studio passes require an Orientation session.<\/p>\n<\/div>\n\n\n\n
